# Integrating Move History from PawnTown

Integrate the move history from PawnTown's chess streams to provide your viewers with a clear and concise record of each game. This guide covers the process for embedding move history and customizing its appearance to suit your platform's design.

## Embed Link

To showcase the move history on your platform, use the following embed link. Remember to replace `[streamId]` with your specific live stream ID:

```html
<iframe src="https://pawn.town/s/[streamId]/embed/moves" width="100%" height="400" frameborder="0" scrolling="no"></iframe>
```

## Customizing Appearance via Query Parameters

You can personalize the appearance of the move history by leveraging the following query parameters:

### 1. Direction (`ltr`):

Set the direction of move display. Use `true` for left-to-right, and `false` for right-to-left.

Example: `https://pawn.town/s/[streamId]/embed/moves?ltr=true`

### 2. Maximum Moves Display (`max`):

Set a maximum limit for the number of moves displayed.

Example: `https://pawn.town/s/[streamId]/embed/moves?max=50`

### 3. Foreground Color (`fc`):

Define the color of the move texts.

You can choose a color value like `green` or hex `%23000000` (=`#000000`).

Example: `https://pawn.town/s/[streamId]/embed/moves?fc=%23000000`

### 4. Background Color (`bg`):

Determine the background color of the move display section.

You can choose a color value like `green` or hex `%23000000` (=`#000000`).

Example: `https://pawn.town/s/[streamId]/embed/moves?bg=%23ffffff`

### 5. Font Family (`f`):

Choose the font family for the move texts.

Example: `https://pawn.town/s/[streamId]/embed/moves?f=Arial`

### 6. Font Size (`s`):

Set the font size for the move texts.

Example: `https://pawn.town/s/[streamId]/embed/moves?s=14px`

### 7. Gap Between Moves (`g`):

Specify the space or gap between individual moves.

Example: `https://pawn.town/s/[streamId]/embed/moves?g=10px`

---

By tailoring the move history appearance to match your platform's aesthetics, you ensure a seamless user experience. Happy streaming with PawnTown!
