# Integrating Player Names from PawnTown

Present your viewers with the names of the competitors in a sleek and customized manner using PawnTown's player name embedding feature. This guide will take you through the steps to embed player names and customize them to fit your platform's aesthetic.

## Embed Link

To showcase the player names on your platform, utilize the following embed link. Be sure to replace `[streamId]` with the specific ID of your live stream:

```html
<iframe src="https://pawn.town/s/[streamId]/embed/names" width="100%" height="60" frameborder="0" scrolling="no"></iframe>
```

## Customizing Appearance via Query Parameters

Enhance the appearance of the player names using the provided query parameters:

### 1. Foreground Color (`fc`):

Choose the color for the player names text.

You can choose a color value like `green` or hex `%23000000` (=`#000000`).

Example: `https://pawn.town/s/[streamId]/embed/names?fc=%23000000`

### 2. Background Color (`bg`):

Set the background color for the player names section.

You can choose a color value like `green` or hex `%23000000` (=`#000000`).

Example: `https://pawn.town/s/[streamId]/embed/names?bg=%23ffffff`

### 3. Font Family (`f`):

Determine the font family for the player names.

Example: `https://pawn.town/s/[streamId]/embed/names?f=Arial`

### 4. Font Size (`s`):

Adjust the font size for the player names text.

Example: `https://pawn.town/s/[streamId]/embed/names?s=16px`

---

By customizing the player names' appearance, you can maintain the consistency and branding of your platform. With PawnTown, you're always in control of the look and feel. Happy streaming!
