# üé¨Ô∏è Embed Full Player

Embedding PawnTown's full player on your platform brings live chess action right to your audience, boosting their experience üéâ. This guide will help you through the setup process and show you how to customize the player to fit your style perfectly üõ†Ô∏è. Let's get started!

## HTML

To integrate the full player, utilize the following embed link, where `[streamId]` should be replaced with the unique ID of your live stream:

```html copy
<iframe
  src="https://pawn.town/s/[streamId]/embed"
  width="100%"
  height="600"
  frameborder="0"
  scrolling="no"
></iframe>
```

## Query Parameters

| Parameter                | Description                                                                                               |
| ------------------------ | --------------------------------------------------------------------------------------------------------- |
| `l` (Light Square Color) | Define the color of the light squares. Can be a color value like `green` or hex `%23000000` (=`#000000`). |
| `d` (Dark Square Color)  | Define the color of the dark squares. Can be a color value like `green` or hex `%23000000` (=`#000000`).  |
| `e` (Show Evaluation)    | Toggle the display of the board's evaluation. Use `true` to show, and `false` to hide.                    |
| `c` (Control Bar)        | Toggle the visibility of the control bar. Use `true` to show, and `false` to hide.                        |

**Example of Using Parameters:**
To integrate a chess board with light gray squares, dark gray squares, evaluation shown, and control bar hidden, your URL would look like this:

```url copy
https://pawn.town/s/[streamId]/embed?l=%23e0e0e0&d=%23202020&e=true&c=false
```
